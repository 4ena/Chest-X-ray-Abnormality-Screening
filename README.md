# Chest-X-ray-Abnormality-Screening

A project that aims to use image data, classify it, then compares it with other patients to rank them based on severity

This is Multi-Abnormalities detection
- Cardiomegaly
- Pleural effusion
- Atelectasis
- Infiltration
- Opacity

Uses the [CheXpert-v1.0-small](https://www.kaggle.com/datasets/ashery/chexpert/data?select=train.csv) image dataset.
